# 🐱 Кошачий пинтерест

Тестовое задание для стажёрской программы Учи.ру — интерфейс для просмотра котиков с возможностью добавлять их в избранное.

**[👉 Смотреть демо](https://hamsike.github.io/frontend-challenge/)**

## 📋 Выполненные требования

### Обязательные
- ✅ По умолчанию открывается вкладка "все котики"
- ✅ У каждого котика есть возможность добавить в "любимые" и убрать из "любимых"
- ✅ Данные о любимых котиках хранятся на клиенте (localStorage)
- ✅ На вкладке "любимые котики" отображаются добавленные в избранное котики
- ✅ Проект опубликован на GitHub Pages

### Дополнительные (плюсы)
- ✅ **Адаптивность** — интерфейс корректно отображается на всех устройствах
- ✅ **Бесконечная прокрутка** — котики подгружаются автоматически при скролле

## 🛠 Использованные технологии

| Технология | Назначение |
|------------|------------|
| **React 18** | Библиотека для построения пользовательского интерфейса |
| **TypeScript** | Типизация кода для надёжности и удобства разработки |
| **Vite** | Быстрая сборка проекта и hot-reload |
| **Redux Toolkit** | Управление глобальным состоянием (избранное) |
| **React Query (TanStack Query)** | Кэширование и пагинация запросов к API |
| **Axios** | HTTP-клиент для запросов к TheCatAPI |
| **CSS Modules** | Изолированная стилизация компонентов |
| **GitHub Pages** | Деплой и хостинг готового приложения |

## 🚀 Запуск проекта локально

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Hamsike/frontend-challenge.git
cd frontend-challenge
```
### 2. Установите зависимости
```bash
npm install
```
### 3. Запустите dev сервер
```bash
npm run dev
```
